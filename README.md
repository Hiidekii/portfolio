# Portfolio Hideki Sotero

¡Hola! Soy **Hideki Sotero**, Systems Engineer graduado de la Universidad de Lima (décimo superior). Este es mi portafolio web personal, donde presento mis proyectos, experiencia y habilidades técnicas en el desarrollo de software.

## 🚀 Tecnologías Principales
Este proyecto está construido utilizando un stack moderno enfocado en alto rendimiento y escalabilidad:

*   **Framework**: React con Vite.
*   **Lenguajes**: TypeScript y JavaScript.
*   **Estilos**: Tailwind CSS y shadcn/ui.
*   **Arquitectura/Herramientas**: Koin (DI) y Room (Database) en mis implementaciones móviles, integradas con los estándares de diseño de mis proyectos anteriores.

*   **Desarrollo Móvil**: Experiencia enfocada en arquitectura, frameworks cross-platform y herramientas como Kotlin, Flutter y Expo.
